# покрытие
Покрытие лежит в [html файле](./coverage/index.html) достаточно также просто открыть в браузере

# для доп проверки
## сборка и запуск обычной работы
1. установить `npm`
2. перейти в дерикторию `Pascal`
3. выполнить команды
```bash
npm install
npm run build
```
4. открыть [index.html](./index.html) в браузере.

## проверка покрытия (тут нужно небольшое взаимодействие с кодом)

1. установить `npm`
2. перейти в дерикторию `Pascal`
3. в [коде](./src/compiler.ts) в 27 и 188 строках перед `class` добавить `export`
4. выполнить команды
```bash
npm install
npm run build
npm run test:coverage
```
5. открыть [html файл](./coverage/index.html) в браузере.
6. если снова надо собрать основной компилятор в [коде](./src/compiler.ts) в 7 строке перед `class` убрать `export`
