# как запустить

поскольку работа с байтами через сокеты в браузере небезепосна сделал код для запуска в консоли

# покрытие
Покрытие лежит в [html файле](./coverage/index.html) достаточно также просто открыть в браузере

# для доп проверки
## сборка и запуск обычной работы
1. установить `npm`
2. перейти в дерикторию `best_program`
3. выполнить команды
```bash
npm install
npm run start
```
4. Данные начнут собираться в файл, для остановки достаточно нажать `Ctrl+C`

## проверка покрытия (тут нужно небольшое взаимодействие с кодом)

1. установить `npm`
2. перейти в дерикторию `best_program`
3. выполнить команды
```bash
npm install
npm run test:coverage
```
4. открыть [html файл](./coverage/index.html) в браузере.
